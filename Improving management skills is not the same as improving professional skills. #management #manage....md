# 提升管理能力，不等於讓專業能力永遠強過下屬

## 管理者不需要在所有專業上都比員工強

如果管理者總是追求自己的專業能力比每一名下屬都強，這個團隊通常走不遠。

人們願意跟隨一個人，常見的原因有兩類：一是認同他的品格與做事方式，二是佩服他的能力與才華。有些管理者因此認為，只有自己的專業能力始終高於員工，才能讓團隊服氣。

但專業能力與管理能力，本來就是兩種不同能力。

## 永遠比下屬專業，會讓主管停留在個人貢獻者角色

要讓專業能力一直比下屬強，管理者就必須投入和員工相同、甚至更多的時間鑽研專業。

這表示他主要仍在扮演資深專業人員，而不是管理者。

當團隊規模擴大、專業分工變細後，主管不可能在每個領域都比所有人厲害。真正需要提升的，是如何讓不同專業的人共同完成目標。

## 員工服從管理，靠的是管理品質

員工是否認同主管，不只看主管會不會做自己的工作，也會看：

- 工作分工是否合理
- 資源配置是否公平
- 目標與優先順序是否清楚
- 能否激發團隊投入
- 遇到問題時是否願意承擔
- 獎勵與責任是否一致
- 決策是否有依據

這些都是管理能力，而不是單純的專業技術。

## 管理者要把時間投入在人與系統

管理者需要學習的內容包括：

- 如何識別不同人的能力與動機
- 如何安排合適的工作
- 如何建立信任
- 如何針對不同需求設計激勵
- 如何讓員工理解並接受管理要求
- 如何在不製造過度負面情緒的情況下推動改變

專業能力仍然重要，因為它能幫助主管理解工作、判斷風險，也能和下屬進行有效溝通。但主管不需要把「每件事都比下屬會做」當成管理資格。

好的管理者，不是團隊裡最強的單兵，而是能讓不同能力的人，在合理分工與清楚規則下共同產生成果。
