# 怎麼讓擺爛的資深員工離開？

## 不要只按員工類型尋找不同招式

有人問：「討厭的員工開不掉，該怎麼辦？」

其實，讓一名員工離開，與他是擺爛、躺平、搗蛋，還是績效不佳，背後的基本邏輯沒有本質差異。

如果每遇到一種類型，就分別研究「這類人怎麼開、那類人怎麼裁」，表示仍停留在表面分類，沒有看見問題的共同核心。

## 人會離開，是因為重要需求無法再被滿足

一個人願意留在組織，通常是因為這裡能滿足某些需求。

就像你刷到一段影片，三秒鐘後發現不是自己想看的內容，自然會滑走。員工也是一樣：如果他在組織裡長期得不到想要的東西，留下的理由就會逐漸消失。

因此，讓一名不合適的員工離開，核心往往是讓原本支持他留下的條件不再存在。

但這必須建立在合法、合規與合理的制度上，不能用羞辱、違法打壓或惡意逼退的方式處理。

## 先辨識他為什麼願意留下

管理者需要了解，這名員工真正重視的是什麼：

- 穩定薪資；
- 工作輕鬆；
- 特定職位或頭銜；
- 離家近；
- 彈性時間；
- 團隊關係；
- 某些資源或便利；
- 缺乏績效壓力。

如果組織重新建立公平的工作要求、績效標準與責任邊界，而這名員工不願意適應，他可能會自行評估是否繼續留下。

## 管理重點應是恢復正常規則

真正合理的做法，不是針對某個人設計陷阱，而是把正常管理制度建立起來：

1. 明確職責與工作要求；
2. 設定可衡量的績效標準；
3. 提供合理改善機會；
4. 保存完整溝通與考核紀錄；
5. 對所有人使用一致規則；
6. 依法處理持續不適任的情況。

當員工必須真正履行職責，而不再能長期享受「不做事也沒有後果」的狀態時，他要麼改善，要麼離開。

## 看見本質，才能換場景仍會處理

如果只記住一張表格或一套針對「老油條」的話術，換成合夥人、主管或其他類型員工時，就又不知道該怎麼辦。

真正有用的管理能力，是看懂人為什麼留下、為什麼離開，以及如何透過公平制度讓不合適的人作出選擇。
